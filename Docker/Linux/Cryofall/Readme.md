# Cryofall Dedicated Server
## How To

##### Open Firewall Ports:
 - 6000/UDP

#### Create Data Location
```
mkdir -p $HOME/cryofall
 ```

#### Run Container
```
cd /path/to/your/docker-compose/file
docker-compose up
 ```

### Locations
 - Server Files: ~/cryofall
 - Settings:  ~/cryofall/SettingsServer.xml
 - Settings:  ~/cryofall/ServerRates.config
